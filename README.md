# CPP-POOL

Ten C++ modules written in C++98 as part of the 42 cursus. The C++98 constraint means no lambdas, no `auto`, no smart pointers, no range-based loops.

## Build

```bash
cd cpp0X/exXX
make
```

## Modules

- **cpp00** - Namespaces, classes, member functions, I/O streams, initialization lists, `static`, `const`.
- **cpp01** - Memory allocation (heap vs stack), pointers to members, references, file streams.
- **cpp02** - Operator overloading, Orthodox Canonical Form, fixed-point arithmetic.
- **cpp03** - Inheritance, constructor chaining, `protected` vs `private`, diamond problem.
- **cpp04** - Polymorphism, virtual/pure virtual functions, abstract classes.
- **cpp05** - Exceptions, `try`/`catch`/`throw`, custom exception classes.
- **cpp06** - C++ casts: `static_cast`, `dynamic_cast`, `reinterpret_cast`, `const_cast`.
- **cpp07** - Function and class templates, template specialization.
- **cpp08** - STL containers (`vector`, `list`, `map`, `stack`), iterators, algorithms.
- **cpp09** - STL applied: `std::stack` for RPN evaluation, `std::deque` for Ford-Johnson sort, `std::map` for Bitcoin exchange rate conversion.
